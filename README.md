## Instalação e uso

```bash
# Clone this repository
$ git clone https://github.com/isaacmarcos-oficial/RocketShoes
# Go into the repository
$ cd react-rocketshoes
# Install dependencies
$ yarn install
# Run the app
$ yarn start
```
